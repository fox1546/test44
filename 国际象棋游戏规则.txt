========================================
         国际象棋游戏规则说明
========================================

一、棋盘说明
---------------
国际象棋棋盘是8x8的正方形网格，由64个交替的深色和浅色方格组成。
- 横向：用字母a-h标记（从白方视角从左到右）
- 纵向：用数字1-8标记（从白方视角从下到上）

棋盘标记示例：
a8 b8 c8 d8 e8 f8 g8 h8
a7 b7 c7 d7 e7 f7 g7 h7
...
a1 b1 c1 d1 e1 f1 g1 h1


二、棋子说明
---------------
国际象棋共有六种棋子，每种棋子有其独特的移动方式：

1. 王 (King) - 标记: K(白) / k(黑)
   - 价值：无限
   - 移动方式：可以向任何方向（横、竖、斜）移动一格
   - 特殊：王被"将军"时必须立即应将；王被将死时游戏结束

2. 后 (Queen) - 标记: Q(白) / q(黑)
   - 价值：9分
   - 移动方式：可以横、竖、斜向移动任意格数
   - 特点：国际象棋中威力最大的棋子

3. 车 (Rook) - 标记: R(白) / r(黑)
   - 价值：5分
   - 移动方式：可以横向或纵向移动任意格数
   - 特殊：可以参与"王车易位"

4. 象 (Bishop) - 标记: B(白) / b(黑)
   - 价值：3分
   - 移动方式：只能斜向移动任意格数
   - 特点：每只象只能在同色的格子上移动

5. 马 (Knight) - 标记: N(白) / n(黑)
   - 价值：3分
   - 移动方式：走"L"形（先走2格，再横走1格；或先横走1格，再走2格）
   - 特点：是唯一可以跳过其他棋子的棋子

6. 兵 (Pawn) - 标记: P(白) / p(黑)
   - 价值：1分
   - 移动方式：
     * 白兵向上移动（数字减小方向），黑兵向下移动（数字增大方向）
     * 首次移动可以走1格或2格
     * 之后每次只能走1格
   - 吃子方式：斜向一格吃子
   - 特殊：到达对方底线时可以升变（变为后、车、马、象之一，默认为后）


三、特殊规则
---------------

1. 王车易位 (Castling)
   - 条件：
     * 王和参与易位的车都没有移动过
     * 王和车之间没有其他棋子
     * 王当前未被将军
     * 王经过的格子和到达的格子都不会被攻击
   - 类型：
     * 短易位（王向王翼移动2格，车越过王到另一侧）
     * 长易位（王向后翼移动2格，车越过王到另一侧）

2. 兵升变 (Promotion)
   - 当兵到达对方底线时（白兵到第8行，黑兵到第1行）
   - 可以升变为：后、车、马、象中的任意一种
   - 本游戏默认升变为后

3. 将军 (Check)
   - 当王处于被对方棋子攻击的状态
   - 将军时必须采取以下三种方式之一应将：
     * 移动王离开被攻击的格子
     * 挡住对方的攻击路线
     * 吃掉攻击的棋子

4. 将死 (Checkmate)
   - 当王被将军，且无法通过任何合法手段解除将军状态
   - 被将死的一方输，对方赢

5. 和棋 (Draw) - 本游戏实现的和棋情况：
   - 无子可动（Stalemate / 逼和）：当前回合的玩家王未被将军，但没有任何合法的移动方式


四、游戏操作
---------------

1. 开始游戏
   - 点击菜单栏：游戏 -> 开始

2. 选择棋子
   - 用鼠标左键点击己方棋子
   - 选中后会显示蓝色边框
   - 可用的移动位置会以蓝色圆点标记
   - 可吃子的位置会以红色边框标记

3. 移动棋子
   - 先点击要移动的棋子
   - 再点击目标位置
   - 上一步移动会以黄色高亮显示

4. 游戏模式配置
   - 点击菜单栏：游戏 -> 配置
   - 可以选择：
     * 双人对战：两名玩家轮流操作
     * 人机对战：玩家执白棋，电脑执黑棋


五、胜利与结束条件
---------------------

1. 将死对方
   - 当一方的王被将死时，游戏结束
   - 弹出消息框显示获胜方

2. 和棋（无子可动）
   - 当前回合玩家没有任何合法移动，但王未被将军
   - 游戏以和棋结束


六、界面说明
---------------

- 棋盘格子颜色：浅色（米黄色）和深色（棕色）交替
- 棋子表示：
  * 白色棋子：大写字母（K, Q, R, B, N, P），白色字体带黑色阴影
  * 黑色棋子：小写字母（k, q, r, b, n, p），黑色字体
- 状态显示：
  * 底部显示当前回合方（白方/黑方）
  * 将军时会显示"将军！"提示
